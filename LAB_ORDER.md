# LAB_ORDER.md

This file stores the canonical course order provided by the student.
It is the planning reference for the landing page order and future lab publication work.
Its presence does not mean that source materials already exist locally.
A lab is only ready to build when its files are available in `ignoredocs/` and, ideally, it also has notes in `lab_drafts.md`.
